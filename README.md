
---

## ✨ Fonctionnalités actuelles

### CV
- Design glassmorphism sur fond sombre
- Hero avec photo, bio scrollable et centres d'intérêt
- Section expériences scrollable
- Grille formations / compétences
- Bandeau nom avec effet typewriter au chargement
- Liens portfolio et contact
- Copie au clic sur l'email et le numéro de téléphone
- Animations d'entrée en cascade au chargement
- Responsive mobile

### Galerie artistique
- Accessible depuis le tag **Drawing**
- Grille scrollable de dessins
- Lightbox au clic pour agrandir chaque image
- Navigation clavier (← → Échap)

### Bibliothèque
- Accessible depuis le tag **Reading**
- Livres en cours avec jauge de progression
- Grille des livres lus avec couvertures
- Lightbox au clic sur une couverture

Ce projet est en évolution constante. Au fil du temps seront ajoutés :

## 🛠 Technologies

- **HTML5** — structure
- **CSS3** — design, animations, responsive (media queries)
- **JavaScript vanilla** — interactions, modales, galerie, bibliothèque

Aucun framework, aucune dépendance externe — uniquement des technologies web natives.
